# IaC Homelab Repository

Infrastructure as Code repository for homelab and multi-cloud deployments.

## Structure

This repository follows a structured approach for managing infrastructure across:
- Proxmox homelab infrastructure
- Multiple cloud providers (DigitalOcean, Hetzner, AWS)
- Scalr's three-tier hierarchy (Account, Environment, Workspace)
- Local testing workflows
- OpenTofu migration support
